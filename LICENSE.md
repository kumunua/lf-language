본 라이선스는 LF 언어, LF 실행기, LF 컴파일러, LF 런타임, LF 기본 패키지, LF 문서, LF 예제, LF 공식 배포 파일에 적용돼요.

LF 를 다운로드하거나 설치하거나 실행하는 경우, 사용자는 본 라이선스에 동의한 것으로 간주돼요. 본 라이선스에 동의하지 않는 경우 LF 를 사용할 순 없어요.

## 정의

"LF" 는 `.lf` 소스 파일, LF 언어, LF 실행기, LF 컴파일러, LF 런타임, LF 기본 패키지, LF 문서, LF 예제, LF 공식 배포 파일을 의미해요.

"사용자 소프트웨어" 는 사용자가 LF 를 이용해 직접 작성, 빌드, 컴파일, 제작한 프로그램, 게임, 앱, 도구, 라이브러리, 콘텐츠를 의미해요.

"제작자" 는 LF 의 제작자인 kumunua 를 의미해요.

## 사용 허가

사용자는 LF 로 만든 사용자 소프트웨어를 자유롭게 만들 수 있어요.

사용자는 LF 로 만든 사용자 소프트웨어를 개인적, 교육적, 비상업적, 상업적 목적으로 사용할 수 있어요.

사용자는 LF 로 만든 사용자 소프트웨어를 자유롭게 배포, 판매, 게시, 공유할 수 있어요.

사용자는 LF 로 빌드된 실행 파일을 자신의 사용자 소프트웨어 일부로 배포할 수 있어요.

사용자가 LF 로 만든 사용자 소프트웨어의 저작권과 소유권은 사용자에게 있어요.

## 제한 사항

사용자는 LF 언어, LF 실행기, LF 컴파일러, LF 런타임, LF 기본 패키지, LF 문서, LF 예제, LF 공식 배포 파일을 수정하면 안 돼요.

사용자는 LF 실행기, LF 컴파일러, LF 런타임, LF SDK, LF 공식 배포 파일을 역컴파일, 디컴파일, 디스어셈블, 리버스 엔지니어링하면 안 돼요.

사용자는 LF 실행기, LF 컴파일러, LF 런타임, LF SDK, LF 공식 배포 파일의 보호 장치, 제한 장치, 검증 장치, 라이선스 표시를 우회하거나 제거하면 안 돼요.

사용자는 LF 실행기, LF 컴파일러, LF 런타임, LF SDK, LF 공식 배포 파일을 수정한 형태로 재배포하면 안 돼요.

사용자는 LF 를 자신의 제품처럼 표시하거나, LF의 공식 배포자인 것처럼 오해하게 만들면 안 돼요.

사용자는 LF 의 이름, 로고, 버전 정보, 제작자 정보, 라이선스 문구를 허가 없이 삭제하거나 변경하면 안 돼요.

사용자는 LF 를 악성 코드, 불법적인 목적, 타인의 시스템을 침해하는 목적, 보안 우회 목적, 권한 없는 접근 목적에 사용하면 안 돼요.

## LF 로 만든 소프트웨어의 배포

사용자는 LF 로 만든 사용자 소프트웨어를 자유롭게 배포할 수 있어요.

사용자가 `lf.exe build-exe` 및 `lx.exe exe` 또는 이에 준하는 LF 공식 빌드 기능으로 만든 실행 파일은 사용자 소프트웨어로 간주돼요.

해당 실행 파일 안에 LF 런타임 일부가 포함되어 있더라도, 사용자는 그 실행 파일을 자신의 사용자 소프트웨어 배포에 필요한 범위에서 배포할 수 있어요.

다만 사용자는 LF SDK, LF 컴파일러, LF 실행기, LF 공식 배포 파일 자체를 별도의 SDK, 컴파일러, 런타임, 개발 도구로 재배포하면 안 돼요.

## 공식 배포 경로

LF 는 [공식 배포 경로](https://github.com/kumunua/lf-language/)를 통해 다운로드하는 것을 권장해요.

공식 배포 경로가 아닌 곳에서 받은 LF 파일은 변조되었을 수 있어요.

공식 배포 경로가 아닌 곳에서 받은 LF 파일로 인해 발생한 문제는 공식 지원 대상이 아닐 수 있어요.

사용자는 LF 를 배포하거나 소개할 때 가능한 한 공식 배포 경로를 안내해야 해요.
